About Me

Hello! My name is E.P.C.A.Dulmin. I am currently a student with a strong passion for software development. My journey into the world of coding began with a curiosity for how things work behind the scenes on the web. This curiosity quickly turned into a passion, leading me to pursue a career as a software developer.

I am continuously learning and expanding my skills in various programming languages and technologies, with a particular interest in web development. I have hands-on experience with HTML, CSS, and JavaScript, and I am always eager to explore new tools and frameworks that can enhance my development process.

My goal is to become a proficient software developer who can create efficient, user-friendly, and visually appealing web applications. I enjoy problem-solving and am always up for a challenge. In my spare time, I like to stay updated with the latest trends in technology, contribute to open-source projects, and work on personal coding projects to further sharpen my skills.

Aside from coding, I have a keen interest in design, which helps me create aesthetically pleasing and functional user interfaces. I believe that good design is crucial in making software more accessible and enjoyable for users.

Projects

Country Search App

The Country Search App allows users to search for information about different countries. Users can search by country name or code to get details such as population, capital city, region, and more. This project utilizes an external API to fetch real-time data, ensuring the information is always up-to-date.

iCalc Number Converter System

The iCalc Number Converter System is a versatile tool that converts numbers between different formats such as binary, decimal, hexadecimal, and octal. This project is built with a user-friendly interface and performs conversions quickly and accurately. It's a helpful utility for students and professionals working with different number systems.

Search Contact System

The Search Contact System is a web application designed to manage and search through contact information. Users can add, edit, and delete contacts, as well as search for contacts based on various criteria such as name, email, or phone number. This system is built to help users efficiently manage their contacts and find information quickly.
